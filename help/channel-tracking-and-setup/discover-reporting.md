# 发现报表 {#discover-reporting}

在[!UICONTROL User Settings]内，用户可以选择将货币从默认货币更改为列出的任何公司货币。 这将更改金额（例如，成本、收入和管道收入）的显示方式，以及三个字母的货币代码。

![在“用户设置”中，用户可以选择更改](assets/multi-currency-1.png)
