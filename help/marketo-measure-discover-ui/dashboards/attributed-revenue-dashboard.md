# 已归因的收入仪表板 {#attributed-revenue-dashboard}

已归因的收入仪表板从重点的角度提供与您的营销活动直接相关的收入。 深入了解您的营销策略如何有助于达成交易。

在归因收入方面，哪些渠道、子渠道或营销活动排名最高？
我们的归因收入总额和归因的已结束交易数是多少？
